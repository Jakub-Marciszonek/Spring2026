# Lindstrom-project
part(1/2) Grid editor - tool for creating a layout of the warehouse. It is going to be based on grid so geometry will be limitted. Exports a file for the Optimalization logic and later display.

part(2/2) Optimalization logic - is set up with file from grid editor, then it takes pick up points as an input then it calculates most optimal route for the batch of items.
